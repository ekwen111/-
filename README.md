Описание программы:Простое приложение для учёта личных расходов с фильтрацией, подсчётом суммы за период и сохранением данных в JSON.

Как использовать:

Запустите expense_tracker.py.
Введите сумму, выберите категорию, укажите дату.
Нажмите «Добавить расход».
Для фильтрации выберите категорию/период и нажмите соответствующую кнопку.
Для подсчёта суммы — выберите даты и нажмите «Подсчитать».
Данные сохраняются автоматически.
Примеры использования:

Добавить расход: сумма — 500, категория — еда, дата — 2026-05-01.
Фильтровать по категории «транспорт».
Подсчитать расходы с 2026-04-01 по 2026-04-30.# -
